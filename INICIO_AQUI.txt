FUNCIONES DISPONIBLES EN JAVASCRIPT
================================================================================

En vylex.html ahora tienes estas 8 funciones:

1. logAuditAction(action, entityType, entityId, field, oldValue, newValue, reason)
   → Registra cualquier cambio en auditoría

2. syncSalesPoint(taskId, vendor, clientName)
   → Sincroniza asignación de vendedor a ClickUp

3. syncImplementationPoint(taskId, implementador, startDate)
   → Sincroniza inicio de implementación

4. syncCancellationPoint(taskId, reason, cancelledBy)
   → Sincroniza cancelación de cliente

5. deleteClientWithDocumentation(clientId, reason, documentation)
   → Elimina cliente (requiere documentación)

6. getAuditLogs()
   → Obtiene todos los registros de auditoría

7. getDeletedClients()
   → Obtiene lista de clientes eliminados

8. syncRolesFromClickUp()
   → Sincroniza consultores vs implementadores

================================================================================
ENDPOINTS DISPONIBLES
================================================================================

Auditoría:
  POST   /api/audit/log              → Registra cambios
  GET    /api/audit/logs             → Obtiene auditoría

Sincronización:
  POST   /api/sync/sales-point       → Sincroniza vendedor
  POST   /api/sync/impl-point        → Sincroniza implementación
  POST   /api/sync/cancel-point      → Sincroniza cancelación

Clientes:
  POST   /api/client/delete-documented  → Elimina con documentación
  GET    /api/client/deleted           → Obtiene eliminados

Roles:
  POST   /api/sales/sync-vendedores  → Sincroniza consultores/impl
  POST   /api/consultores/sync       → Sincroniza consultores
