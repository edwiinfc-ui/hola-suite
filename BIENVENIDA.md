## ⚡ SOLUCIÓN RÁPIDA (2 MINUTOS)

### El problema que reportaste:
```
❌ Error de conexión / endpoint
Failed to fetch desde: https://wispro.holasuite.com
```

### La solución:
1. ⚙️ Ve a **Configuración** en el dashboard
2. 💬 Busca **"¡Hola! / Opa Suite API"**
3. 📝 Ingresa:
   - **URL Base:** `https://wispro.holasuite.com`
   - **Token:** [Tu token Bearer]
4. 💾 Haz clic en **"Guardar"**
5. 🔌 Haz clic en **"Probar"**
6. ✅ Verás: "✅ Conectado · /atendimento"

**¡Listo!** Tus conversaciones de Opa aparecerán.

## 💡 TIPS

### Tips para conectar:
- ✅ La URL es: `https://wispro.holasuite.com` (SIN `/api/v1`)
- ✅ El token debe empezar con `Bearer ` (el sistema lo agrega)
- ✅ Si no tienes token, contacta tu admin
- ✅ El Workspace ID es opcional (déjalo vacío)

### Tips si no funciona:
- 🔍 Abre F12 (DevTools) → Console
- 📍 Ahí verás detalles del error
- 📧 Si contactas soporte, comparte ese error
- ⏱️ Intenta en 5 minutos (servidor puede estar ocupado)
